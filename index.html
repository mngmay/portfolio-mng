Résumé

Lambda Treasure Hunt

A cohort wide competition of 100+ students to find the most treasure. Competitors had to traverse an unknown maze, find secret rooms, unlock hidden powers, decode binary messages, and mine blockchain coins to meet various objectives.

This project was a 3.5 day build and challenged my team to develop various algorithms inorder to generate a map and automate a variety of functionality. This project is undergoing continued improvements with the goal of creating a GUI.

Awards: 1st place team for Gold and Golden Snitches
(3 total categories: Gold, Golden Snitches, and Lambda Coins)

  • Role: Frontend Developer
  • Tech Stack: React with Hooks and Context API
  • GitHub Codebase

Pizza Party

An interactive adventure game where users can navigate their turtle adventurer in search of pizza.

This project was a 4 day build and challenged my team to create a maze of interconnected rooms and GUI for interactive play. I focused on frontend design, including calling backend data in order to render a visual map of the dungeon. Users get real time updates on their moves and notifications when they successfully find a slice of pizza!

  • Role: Frontend Developer
  • Tech Stack: React with Hooks and Context API
  • GitHub Codebase
